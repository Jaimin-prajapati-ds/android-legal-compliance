Universal Requirements

Mandatory for all applications published on Google Play.

Provide public Privacy Policy URL Must name the developer entity exactly as it appears in Play Console and list data practices. Action: Enter stable URL in Play Console App Content.
Fill out Play Console Data Safety form Must declare all collected and shared data types (including those collected by third-party SDKs). Action: Complete questionnaire in Play Console.
Verify target SDK version New apps and updates must target API 34 (Android 14) or higher (API 35/36 by late 2026). Action: Update targetSdk in build.gradle.

User Accounts and Data Deletion

Required if users can register or log in.

Implement in-app account deletion A direct option inside the application settings or profile page allowing users to trigger account deletion. Action: Add deletion button and call backend delete functions.
Implement web account deletion link A working website address where users can request deletion of account and data without needing to install the app. Action: Deploy web form and register URL in Play Console.

EEA, UK, and Switzerland (GDPR)

Obligations when distributing to users in Europe.

Establish legal basis for processing Ensure every personal data collection is backed by consent, contractual necessity, or legitimate interest. Action: Document legal bases in Privacy Policy.
Implement user rights request channel Provide a mechanism for users to request data export, correction, or restrictions. Action: Set up email address or request portal.

India (DPDP Act)

Obligations when distributing to users in India.

Maintain data processing logs for 1 year Under DPDP Rules 2025/2026, processing fiduciaries must retain backend transaction and processing logs for at least 1 year. Action: Configure cloud log retention to 365 days.
Appoint a Grievance Officer Designate a grievance redressal contact name and publish it in the Privacy Policy. Action: Add Grievance Officer details to Privacy Policy page.

United States (CCPA/CPRA and State Patchworks)

Obligations when distributing to users in the United States.

Support Opt-Out of sale or sharing Allow users to opt-out of data sharing for targeted advertising. Action: Add a "Do Not Sell My Personal Information" link or toggle.

Ad Monetization

Required if displaying advertising.

Integrate Google-certified CMP (UMP SDK) Collect certified consent for ads in the European Economic Area. Action: Implement User Messaging Platform SDK on app launch.

Children's Privacy (COPPA & Play Families Policy)

Obligations if the application target audience includes minors.

Configure COPPA Request Settings Tag ad and analytics requests directed to children under 13. Action: Call setTagForChildDirectedTreatment on AdMob initialization.
Verify approved SDKs Ensure all integrated third-party libraries are certified on the Play Store Families Policy index. Action: Review Play Store approved list.

Security Baseline

Required to satisfy general security clauses.

Enforce HTTPS traffic Verify cleartext HTTP traffic is blocked in production manifests. Action: Set cleartextTrafficPermitted to false in Network Security Config.
Encrypt local storage sensitive data Use Jetpack EncryptedSharedPreferences for credentials and tokens. Action: Replace standard SharedPreferences references.
